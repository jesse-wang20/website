Explore X

This is our final software engineering project. Below is the final document which consists of the entire process my team and I worked through.

Tools Used

I utilized various tools and technologies throughout the development of Explore X. Following the Software Development life cycle, my team started with a discovery stage. We then proceeded to creating a System Analysis, Use Case Model, User Interface Model, Data Model, and Software Model. Some of the key tools utilized during the process are:

  • Version Control: Git
  • Project Management: Monday
  • User Interface Model: Figma
  • Software Modeling: Lucid Charts
Software Build Tools:
  • Programming Languages: Javascript
  • Framework(s): Express, React
  • Database: MongoDB (Mongoose)
  • Tools/Libraries: Bcrypt, Crypto, Google Oauth2

Project Overview

In this project, I collaborated with three other team members to achieve our application. The main features of Explore X I personally worked on are:

  • User Authentication: Login, Register, Logout, Login as Guest, Forgot Password
  • Front End Development: Design of User Authentication, FAQ section
  • Full-Stack Development: Forking Maps, Saving/Publishing Edited Map
